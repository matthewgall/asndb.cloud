Welcome to asndb

Need information from BGP for your application? asndb.network is for you!

ip2asn: IP address to autonomous systems number

ip2asn is our IP address to autonomous systems number mapping dataset, updated hourly

to pull the latest data (in json format):

curl https://asndb.network/get/latest/ip.json

to pull the latest data (in dat format):

curl https://asndb.network/get/latest/ip.dat

 

asn2name: autonomous systems number to netname

asn2name is our autonomous systems number mapping to netname dataset, updated 6 hours

to pull the latest data (in json format):

curl https://asndb.network/get/latest/asn.json
